Cara Flash Andromax AD686G atau Andromax C

#Program #Tutorial #Info #Networking #Android

Selamat weekend gan and sist sekalian
Weekend gabut mending liat tutor dari ane ini
Flash andromax c atau ad686g series ini tergolong sangat mudah karena tidak memakan waktu yang banyak dan ribet

Bahan bahan yang harus disiap kan 
  1. HH nya lah pasti gan 
  2. Memory Card min 1 GB 
  3.  Frimware 
Cara nya gimana?
  1. Pindahkan semua data di MC anda untuk menghindari suatu kesalahan yang tidak diinginkan
  2. Format MC nya karena harus keadaan kosong (recomend)
  3. Extra file hasil download tadi dan copi folder "EG902.v1" ke dalam sd card anda Ingat file harus diluar folder jangan didalam folder  
  4. Setelah itu pasang sd cardnya kedalam smartfren yg hendak di exekusi lalu pasang batre 
  5. Tekan tombol power + vol up + vol down secara bersamaan sampe proses flashing selesai
  6. Jika ada tulisan sukses tekan tombol power kembali untuk menghidupkan HH anda
  7. Selamat HH anda berhasil diflash

Bahan-bahan yang harus disiapkan : 1. Memorycard external 2. Cardreader (alat untuk menghubungkan memory ke komputer anda untuk pemindahan data. 3. Firmware smartfren andromax c ad686g JB download DI SINI dan DISINI Sebelum mulai flashnya yang harus diperhatikan : 1.Memory external kapasitasnya 1gb atau lebih. 2.Pindahkan semua file di memory ke komputer anda terlebih dahulu dan lakukan format di komputer cara ini agar menghindarkan adanya virus di memory anda. Berikut cara flash smartfren andromax c ad686g JB : 1.Extract firmware yang anda download tadi dengan cara klik kanan pada firmware lalu extratct here 2.Setelah di extract copy EG902.v1 dan paste ke memory external anda ingat jangan di taruh di folder lagi 3.Matikan HP anda atau copot batre agar hp benar-benar dalam keadaan mati , lalu copot simcard , 4.Masukan memorycard external anda yang berisi firmware eg902.v1 ke hp anda , 5.Dalam keadaan HP mati tekan dan tahan tombol volume down + volume up + power on/off sampai muncul proses flash nya lalu lepas smua tombol yang anda tekan tadi lalu tunggu sampai ada tulisan sukses, jika sukses tekan tombol power untuk menghidupkan hp anda. 6.Selamat hp anda kembali sembuh dari penyakitnya , hhee :) Jika ada kesulitan atau masalah saat flash jangan ragu untuk menanyakan karna saya akan membantu sebisa saya. Nb : Untuk yang terjadi masalah "Only new version allowed to upgrade" Coba ganti file yang bernama AUTOPROG.txt , file download DI SINI

  1.  Jika terkendala di "Only New Version  allowed to upgrade" Coba ganti file yang nama nya AUTOPROG.txt dengan file disini
  2. Keluar tulisan Failed to open file/mmc1/eg902.v1/CONF/ad686g.xml with erro no -1 didalam file CONF yang ag902.xml dirubah jadi ad686g.xml
  3. Terkendala bila belum selesai dan error "emmc last write error" itu harus diganti Romo Modo IC Flash nya karena IC nya sudah korup tidak bisa di gunakan lagi
