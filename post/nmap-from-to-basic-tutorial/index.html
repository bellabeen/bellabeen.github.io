NMAP From To Basic Tutorial

#Tutorial #Security #Networking

Вы знаете о NMAP?
Или кто-то уже его использует?
По случаю в этот раз я буду предоставлять базовые инструменты nmap
Nmap - это инструмент с открытым исходным кодом для проведения разведки и проверки сетевой безопасности. Он предназначен для быстрой проверки больших сетей, хотя он также может работать против одного хоста. Nmap использует сложные IP-пакеты сложным способом, чтобы определить, какие хосты доступны в сети, какая служба (имя и версия приложения) предоставляется, какая операционная система (и версия) используется, какой тип брандмауэра / фильтра пакетов используется и ряд других характеристик

Может быть, здесь вы можете скачать инструменты nmap здесь

    Установите Winpcap версии 2.1 или новее, то есть WinPcap_3_0.exe
    перезагружать
    Загрузите последнее приложение Nmap с сайта www.nmap.org,
    Разархивируйте файл с помощью Winzip или любой другой утилиты декомпрессии.
Существует несколько дистрибутивов linux, которые уже включают в себя инструменты nmap

Возможно, это базовая команда, использующая nmap

1. Проверка открытых портов
nmap <host / target IP>
2. Проверьте конкретный порт на целевой машине
nmap -p <host / target IP>
3. Проверка работы службы на порту
nmap - sV <host / target IP>
4. Проверьте порт конечной машины в 1 сегменте сети
nmap <host / target IP>
5. Определите операционную систему машины

    nmap -O <host / target IP>
