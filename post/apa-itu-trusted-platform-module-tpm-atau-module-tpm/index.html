Apa itu Trusted Platform Module (TPM) atau Module TPM

February 14, 2019


Sedikit cerita beberapa hari yang lalu gua dual boot Windows 10 dengan Parrot , cuman itu sangat menyebalkan tidak berjalan seperti biasa nya saat instalasi dual boot pada umum nya. Kenapa? ternyata di laptop gua terpasang sebuah module yang sangat secure
Yaps benar sekali module tersebut adalah

Trusted Platform Module (TPM)

Lohh apa itu TPM? TPM  merupakan module/chip khusus buatan https://trustedcomputinggroup.org/work-groups/trusted-platform-module/ pada perangkat/device tertentu yang menyimpan kunci enkripsi RSA khusus untuk sistem host untuk otentikasi perangkat keras. Itu sangat menyebalkan gaess tpm ada dual jenis ada yang berupa fisik nya ada juga yang berupa hanya software nya saja

Sepengetahuan gua ada beberapa laptop sekarang yang sudah terpasang module TPM contoh nya laptop saya Acer Swift 3

Setiap chip TPM berisi pasangan kunci RSA yang disebut Endorsement Key (EK). Pasangan ini dipertahankan di dalam chip dan tidak dapat diakses oleh perangkat lunak. Storage Root Key (SRK) dibuat ketika pengguna atau administrator mengambil kepemilikan sistem. Pasangan kunci ini dihasilkan oleh TPM berdasarkan Kunci Pengesahan dan kata sandi yang ditentukan pemilik.

Kunci kedua, disebut Attestation Identity Key (AIK) melindungi perangkat terhadap firmware dan modifikasi perangkat lunak yang tidak sah dengan membuat bagian-bagian penting dari firmware dan perangkat lunak sebelum dijalankan. Ketika sistem mencoba untuk terhubung ke jaringan, hash dikirim ke server yang memverifikasi bahwa mereka cocok dengan nilai yang diharapkan. Jika salah satu komponen hash telah dimodifikasi sejak terakhir dimulai, pertandingan akan gagal, dan sistem tidak dapat memperoleh entri ke jaringan.

Chip TPM dapat digunakan dengan sistem operasi utama dan berfungsi paling baik bersama dengan teknologi keamanan lainnya seperti firewall, perangkat lunak antivirus, kartu pintar, dan verifikasi biometrik.

Windows 10 sudah support module tpm  https://docs.microsoft.com/en-us/windows/security/information-protection/tpm/trusted-platform-module-overview module ini juga  terintegrasi dengan BitLocker Security
