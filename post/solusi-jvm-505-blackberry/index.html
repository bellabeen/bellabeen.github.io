Solusi JVM 505 Blackberry

#Tutorial #Info #Internet #Download


Pernah kah saat anda menghidupkan blackberry anda muncul peringatan JVM Error 505?
Anda sudah berkali kali merestart terus mencabut batrai tapi hasil nya nihil begitu saja tidak ada yanng berubah?

JVM Error 545 adalah JVM Error yang paling buruk karena melumpuhkan perangkat BlackBerry Anda,ada banyak error jvm di blackberry tapi pada kesempatan kali ini saya akan memberikan solusi error 505 :D

Pada dasarnya  JVM Error 545 berarti bahwa Java Virtual Machine yang berfungsi untuk menjalankan software berbasis yang java di aplikasi pada BlackBerry telah mengalami masalah,mungkin itu crash atau yang lainnnya,JVM Error sendiri ada bermacam-macam, ada JVM Error 102, 505, 517, dan 545. Intinya semua error karena Java perangkat Anda bermasalah

Jadi bagaimana cara memperbaiki nya?

Oke kita cek cara nya

Bahan nya
  1. Blackberry Desktop cari di web nya yee :p
  2. Koneksi internet gan pasti nya hhehe
Nih begini

  1. Buka software Blackberry Desktop Manager yang telah anda download tadi
  2. Koneksikan Blackberry ke komputer dengan menggunakan kabel USB (recommend make usb bawaan) 
  3. Saat menghidupkan hh cabut baterai Anda dari perangkat, kemudian klik application 
    
 4. Keluar peringatan nanti dilayar anda Ada tulisan Retry, Update dan Cancel setelahnya yang menanyakan apakah Anda ingin  melakukan update terbaru untuk OS BlackBerry Anda. Klik Update. (versi bb desktop mempengaruhi tapi intinya sama) 

    
5. Setelah itu muncul kotak email, isilah email Anda. Kemudian proses update akan dijalankan, waktunya cukup lama sekitar 45-60 menit.


 6. Pada saat pertengahan update, akan keluar tulisan “Application Loader was unable to connect with your device. Please re-connect your device, enter the password if required, click retry ” seperti pada gambar di bawah. Jangan panik! inilah saat Anda memasang baterai Anda kembali, kemudian tunggu sampai perangkat menyala, klik retry.


7.Setelah itu tunggu proses reboot, error JVM 545 sudah diperbaiki, namun BB Anda akan menjadi seperti baru karena factory setting.


8. Jika Anda membackup data Anda, restore kembali data Anda maka BB Anda akan kembali seperti semula,kalau anda lupa membackup itu resiko ditanggung pemenang yak :D

Mau copas ye? Hargai penulis yaa cantumkan sumber ini :p
