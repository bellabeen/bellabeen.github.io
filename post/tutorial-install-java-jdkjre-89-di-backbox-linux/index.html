[tutorial] Install Java JDK/JRE 8/9 di Backbox Linux

August 30, 2018

Java-bellabeen


Hallo selamat pagi semua nya pada kesempatan kali ini gua akan memberikan tutorial menginstall Java JDK 8⁄9  di Backbox Linux, karena documentasi pada website oracle.com masih kurang jelas. Saya berinisiatif untuk menginstall via apt-get 

Daripada banyak cincong mending kita langsung aja install

Instalasi Default Java JDK/JRE


Pertama kita update terlebih dahulu backbox linux nya
sudo apt-get update
Lalu install JRE (Java Runtime Environment)
sudo apt-get install default-jre
Lalu install JDK (Java Development Kit)
 sudo apt-get install default-jdk 

Selain menggunakan Default Java JDK/JRE bawaan linux kita juga bisa install via repository nya Oracle

Pertama tambahkan Repo PPA Oracle
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
Lalu install Oracle Java JDK 8
sudo apt-get install oracle-java8-installer
Atau bisa juga versi terbaru yaitu JDK 9
 sudo apt-get install oracle-java9-installer

Untuk mengkonfigurasi java agar bisa di gunakan banyak server dengan command
sudo update-alternatives –config java
hachi@geek:~$ sudo update-alternatives –config java
[sudo] password for hachi:
There are 2 choices for the alternative java (providing /usr/bin/java).

  Selection    Path                                            Priority   Status
————————————————————
* 0            /usr/lib/jvm/java-8-oracle/jre/bin/java          1081      auto mode
  1            /usr/lib/jvm/java-8-openjdk-amd64/jre/bin/java   1081      manual mode
  2            /usr/lib/jvm/java-8-oracle/jre/bin/java          1081      manual mode

Press <enter> to keep the current choice[*], or type selection number:

Tekan saja 0 untuk default nya

Lanjut ketik

sudo update-alternatives –config javac

Setting the JAVA_HOME Environment Variable

Agar kita bisa mengeksekusi program java dengan command yang sama kita harus configurasi terlebih dahulu dengan perintah

sudo update-alternatives –config java
Copi direktori yang kamu tuju untuk instalasi lalu buka direktori /etc/environment lalu gunakan text editor favorit kalian semua
 sudo nano /etc/environment
Dan pada akhir file environment tersebut kita tambahkan ini
JAVA_HOME=“/usr/lib/jvm/java-8-oracle”
ataupun ini  sesuaikan dengan versi yang kalian install tadi
JAVA_HOME=“/usr/lib/jvm/java-9-oracle” 

Save dan exit ,lalu reload file tersebut dengan perintah
source /etc/environment
 Kamu bisa mengetes apakah java nya sudah berjalan atau belum kita bisa gunakan perintah

echo $JAVA_HOME
 Kita juga bisa menggunakan perintah java -version untuk mengetahui kita menggunakan java versi berapa

hachi@geek:~$ java -version
java version “1.8.0_181”
Java™ SE Runtime Environment (build 1.8.0_181-b13)
Java HotSpot™ 64-Bit Server VM (build 25.181-b13, mixed mode)
hachi@geek:~$
