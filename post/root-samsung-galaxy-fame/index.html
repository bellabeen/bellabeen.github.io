Root Samsung Galaxy Fame

#Tutorial #Info #Internet #Android #Download

 
Libur telah berakhir semua nyaa :D
Waktu nya kembali ke sekolah lagi :(
Dapet bm dari temen lama nih,dia udah berkali kali ngeroot tapi gagal,nah kok bisa?
Bisa saja gan kemungkinan dalam metode ngeroot dalam android itu sangat besar,bisa aja faktor exploit yang dilakukan aplikasi root tersebut ,atau memang belum ada yang buat exploit buat ngeroot nya :D
Hehehe,dia sudah coba metode menggunakan Framaroot tapi mendapatkan kegagalan dikarenakan framaroot banyak melakukan exploit nya
Disini saya menggunakan Key Root Master
Tutorial nya seperti ini
  1. Downlod Key Root Master  disini ini
  2. Instal aplikasi itu seperti biasa
  3. Misalkan tidak diizinkan centang Sumber Aplikasi lain di pengaturan
  4. Aktifkan USB Debugging, pergi menuju menu settings > Developer options. Untuk android versi4.2 ke atas pergi ke setting > about phone > tekan build number berkali-kali sampai muncul pemberitahuan menjadi pengembang, kembali ke setting > developer option.
  5. Buka aplikasi, tekan tombol Mulai Root.
  6. Jangan panik dikarenakan menggunakan bahasa yang sulit dimengerti :D
  7. Nanti keluar menu pop root ,tekan root
  8. Nanti keluar memberitahuan lagi,Tekan tombol batal (pilihan warna ungu). Bila nampak pemberitahuan cloud root berarti tak dukungan piranti atau exploit nya tidak cocok

 Selesai seperti itulah cara ngeroot Samsung Galaxy Fame buat anda semua nya
