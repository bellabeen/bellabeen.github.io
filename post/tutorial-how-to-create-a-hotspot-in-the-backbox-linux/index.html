[tutorial] How To Create a Hotspot in the Backbox Linux

September 18, 2018


Yo good night all, on this occasion I will give you a tutorial on how to make an Access Point or Hotspot on a personal laptop, or on an OS Backbox Linux, Debian, Ubuntu. The software that we use on this occasion is create_ap

Instead of being confused about how better we look at the tutorial below
Install the packet below

sudo apt-get install dnsmasq haveged hostapd build-essential


Download create_ap packet

git clone https://github.com/oblique/create_ap

cd create_ap

sudo make install
 
We run create_ap service


create_ap start service

To check the status run:
create_ap status service

Using the systemd features (it is not available on Ubuntu 14.04):
 
sudo systemctl start create_ap.service
sudo systemctl enable create_ap.service

Start making Access Points

To create your access point run:

     Check which device will be created as ap with the ifconfig command
     Then we run create_ap

sudo create_ap wlan0 wlan0 MyAccessPoint MyPassPhrase

  wlan0 wlan0 is a wifi adapter found on your PC
  MyAccessPoint is the access point name you created
  MyPassPhrase is a password that you will need to connect to AP * you don’t have to make sure you have 8 characters
