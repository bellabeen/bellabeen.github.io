[tutorial] Install Gambas on Backbox Linux

#Tutorial #Ubuntu #Backbox



Setelah beberapa waktu yang lalu gua udah ngasih tau IDE programming Visual Basic di Linux , pada kesempatan kali ini gua juga akan memberikan tutorial instalasi IDE MonoDevelop lain nya seperti Visual Basic di Wedus

Gambas? Terdengar aneh kata kata tersebut . Yapss gambas merupakan salah satu dari banyak nya software pengembang Visual Basic di BackboxLinux. Gambas juga termasuk pengembangan software based on interpreter dengan sebuah object yang akan dibuat sangat mudah digunakan nya dan mirip seperti Visual Studio di Wedus

Bagaimana sih kak intalasi nya?
Oke kita mulai yukk

Pertama add PPA on your Repo

sudo add-apt-repository ppa:gambas-team/gambas3

Update source and install

sudo apt-get update
sudo apt-get install gambas3
Mudah kan?
Ini tampilan gambas pada laptop gua
