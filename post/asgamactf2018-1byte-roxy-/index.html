[ASGamaCTF2018] 1byte ROXy ?

#kripto #writeups #CTF #asgama2018 #asgamactf

Kategori : Kripto
Soal : 1byte ROXy ?

Diberikan hasil enkripsi xor e7c1cdc1e3f4e6dbcfcec5ffc2d9d4c5ffd8cfd2d29f9fdd
Disini kita harus bruteforce hasil enkripsi tersebut agar mendapatkan flag

Kita gunakan script python
import sys
import base64
a = “e7c1cdc1e3f4e6dbcfcec5ffc2d9d4c5ffd8cfd2d29f9fdd”
b1 = a.decode(“hex”)
for a in range (1,256):
    for b in b1:
        c = ord(b)
        d = c^a
        sys.stdout.write(chr(d))
    print “ Kunci “+str(a)+”\n”  
Pada line 160 didapatkan flag
