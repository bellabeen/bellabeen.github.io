[cpp] Cara Membuat File Header Include C++ Geany Linux

#cpp #Tutorial

Pada kesempatan kali ini saya akan memberitahukan bagaimana cara nya membuat custom header file pada geany untuk bahasa pemerograman cpp
Karena disini saya menggunakan linux, langsung saja kita cek

Disini kita gunakan define  untuk mendefinisikan variable apa yang akan di gunakan dan diganti, nanti kita save berekstensi kan .c contoh waktu.c

Disni saya akan membuat custom header untuk mengkonversi waktu dari jam ke menit dan detik atau sebaliknya
Kita lihat contoh penggunaan nya

#define start() main ()
#define print cout
#define jam (h)
#define jamkemenit (h*60)
#define jamkedetik (h*3600)
#define menitkejam (m/60)
#define menitkedetik (m*60)
#define detikkejam (s/3600)
#define detikkemenit (s/60)
Keterangan
Bisa kita lihat kita ubah penggunaan cout menjadi print
Dan main() menjadi start()
lalu kita definisikan untuk
jamkemenit (h*60) -> untuk langsung menggunakan rumus dalam pembuatan program nya
dll bisa di bilang , kita menyederhanakan function pada program dengan mendefinisikan terlebih dahulu di header nya dan menggunakan konsep ADT dalam pengerjaan nya
 

Kurang lebih seperti itu
Untuk program konversi waktu menggunakan konsep UDT next time kita bahas
