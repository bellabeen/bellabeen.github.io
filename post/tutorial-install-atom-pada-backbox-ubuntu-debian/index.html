[tutorial] Install Atom pada Backbox, Ubuntu, Debian

#Tutorial #Ubuntu #Networking #Internet #Terminal #packet #Backbox

Pada kesempatan kali ini saya akan memberikan tutorial install atom editor pada backbox
Kalian bisa menginstall atom pada linux khusus nya backbox melalui terminal dengan cara

hachi@geek:~$ curl -L https://packagecloud.io/AtomEditor/atom/gpgkey | sudo apt-key add -
Setelah selesai kita bisa menggunakan  perintah

hachi@geek:~$ sudo sh -c ‘echo “deb [arch=amd64] https://packagecloud.io/AtomEditor/atom/any/ any main” > /etc/apt/sources.list.d/atom.list’
Setelah sudah kita update backbox kita

hachi@geek:~$ sudo apt-get update

Kamu bisa download dari terminal dengan perintah apt-get

# Install Atom 
hachi@geek:~$ sudo dpkg -i atom-amd64.deb

# Install Atom’s dependencies if they are missing 
hachi@geek:~$ sudo apt-get -f install
Atau mendownload file .deb di link https://atom.io/download/deb
 Perintah diatas digunakan untuk menginstall atom pada ubuntu debian,backbox
Tapi tidak bisa digunakan pada Red Hat and CentOS (YUM), or Fedora (DNF)
Sekian tutorial singkat semoga bermanfaat
