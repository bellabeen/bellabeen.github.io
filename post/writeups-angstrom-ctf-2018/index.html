[writeups] ANGSTROM CTF 2018

April 1, 2018

 

Pada tanggal 16 - 22 Maret 2018 telah berlangsung kompetisi CTF yang berada di website http://angstromctf.com. Pada saat itu kami berhasil menyelesaikan beberapa soal yang telah kami solved berikut beberapa soal dan write ups nya

1.Warm Up (10)
Soal Diberikan website  yang mengarah ke link https://angstromctf.com/static/crypto/warm_up/ciphertext.txt
Pada file tersebut terdapat file yang berisi myjd{ij_fkwizq}. Kita buka web cryptii.com. Pertama kita gunakan Caesar Chiper ,tapi tidak mendapatkan apa apa, Maka pada clue nya yaitu A very Fine cipher,kita ganti menjadi Affine Cipher maka kita decode hasil cipher tadi menggunakan Affine Chiper dengan Slope A 71 Intercept B 45 maka Wala mendapatkan flag nya actf{it_begins}


 2. Back To Base-ICS(20)
Soal mengarah pada https://angstromctf.com/static/crypto/back_to_base-ics/problem.txt yang berisi 4 bagian flag tersebut Part 1 berisi sebuah binary code yang akan di decode menggunakan website https://www.binaryhexconverter.com/binary-to-ascii-text-converter dan mendapatkan flag actf{0ne_tw0_f0
Part 2 berisi code Octa yang apabila di decode menggunakan web http://www.unit-conversion.info/texttools/octal/ akan mendapatkan flag ur_eight_sixt33
Part 3 berisi code Hexa yang apabila di decode menggunakan web https://cryptii.com/ menghasilkan flag n_th1rtytw0_s1x
Part 4 berisi base64 code yang akan di decode menggunakan website https://www.base64decode.net/ akan menghasilkan potongan flag tyf0ur_no_m0re}
Apa bila di gabungkan menjadi actf{0ne_tw0_f0 ur_eight_sixt33 n_th1rtytw0_s1x
tyf0ur_no_m0re}

3. IRC (10)
Soal berisi kalimat yang berisikan bahwa kita bisa mendapatkan flag apabila mengunjungi channel irc nya di Channel #angstromctf maka kita buka website http://webchat.freenode.net/ dan mendapatkan flag actf{irc}

4. Waldo 1 (10)
Pada soal ini kita di arahkan untuk mendownload file di https://angstromctf.com/static/misc/waldo_1/flags.zip pada file tersebut berisi beberapa foto bendera negara,setelah diamati pada bendera Rusia terdapat flag nya yang berisi actf{there_is_nocollusion(between_teams)}

5.  Waldo 2 (20)

Soal yang didapatkan pada website https://angstromctf.com/static/misc/waldo_2/waldo.zip yang berisi 500 file foto yang bentuknya sama semua,tapi ada satu file yang tidak bisa dibaca file nya. Maka saya ganti ekstensi file tersebut dengan .txt maka mendapatkan file nya actf{r3d_4nd_wh1t3_str1p3s}
6. Thats Not My Name (40)

Pada file yang didownload di link https://angstromctf.com/static/misc/not_my_name/gettysburg.pdfberisi sebuah file pdf yang apabila dibuka tidak bisa
Pada clue selanjutnya di judul nya That’s Not My Name kemungkian ekstensi file tersebut bukan file tersebut. Jadi saya berasumsi bahwa  file tersebut harus diganti ekstensi menjadi doc atau docx
Maka di dapatkanlah flag nya  actf{thanksmrlincolnbutwhoevenuseswordanymore}

7. File Transfer (40)
Soal yang mengarah pada link https://angstromctf.com/static/misc/file_transfer/capture.pcap yang berisi file capture.pcap . Saya yakin kita harus menggunakan wireshark untuk membuka file tersebut. Maka kita analisa file tersebut
Maka kita export data tersebut maka didapatkan file berisi bin yang gak bisa dibaca. Maka kita ganti ekstensi file tersebut menjadi .jpeg dan kita mendapat flag nya actf{0ver_th3_w1r3}

8. GIF (50)
Pada file yang bisa didownload di https://angstromctf.com/static/misc/gif/jiggs.gif tapi entah kenapa file tersebut tidak bisa bergerak,saya berasumsi ekstensi file tersebut salah
Maka saya ganti ekstensi file tersebut menjadi .mp4

9. RUN ME (20)
Pada soal kali ini kita diharuskan mengakses ssh pada server menggunakan account yan telah di sediakan. Setelah login maka kita harus mencari di dalam directori /problems/run_me
Kita harus meneksekusi file tersebut dengan command ./run_me maka walla kita ketemu flag nya actf{why_did_you_run_me}

10. REV1(60)
Kita masih mengeksekusi shell ssh tersebut kita masuk ke direktori /problems/rev1/ terhadapat 
file flag maka kita string file tersebut pada file rev 1_32 maka mendapatkan password nya   
s3cret_pa55word.Maka kita eksekusi file nya dengan perintah ./rev1_32 lalu masukkan password tadi dan bom kita dapat flag nya actf{r3v_is_just_gettin_started!)


11. Source Me 1 (20)
Pada soal ini kita diarahkan kepada website ini http://web.angstromctf.com:6999/ maka kita harus login. Bagaimana kita login? Dengan kekuatan yang besar saya menggunakn kombinasi ctrl+u untuk melihat source code nya didapatkan password f7s0jklKita masukkan username = admin password = f7s0jkl. Maka kita dapat flag nya actf{source_aint_secure}
 
12.GET ME (30)
Soal yang mengarah pada link http://web.angstromctf.com:3005/ jika kita submit akan menghasilkan Hey, you’re not authorized! Pada url terdapat query auth=false , saya iseng mengganti query false tersebut menjadi true maka kita dapatkan flag nya actf{why_did_you_get_me}

13. SOURCE ME 2 (50)
Pada soal ini kita diarahkan menuju link http://web.angstromctf.com:7000/ yang terdapat halaman login kita harus memasukkan username dan password ,kita analisa pada source code nya terdapat enkripsi md5 bdc87b9c894da5168059e00ebffb9077. Maka kita decode md5 tersebut di link hashkiller .
Kita masukkan username admin password = password1234 Kita dapatkan flag nya actf{md5_hash_browns_and_pasta_sauce} 

14. SEQUEL
Pada soal ini kita mendapat hint yaitu SQL ,kita mengarahkan ke website  http://web2.angstromctf.com:2345/  Pada file tersebut kita menggunakan query sql ‘ or “=’ untuk username dan password ‘ or “=’
Maka kita dapat flag nya actf{sql_injection_more_like_prequel_injection}
