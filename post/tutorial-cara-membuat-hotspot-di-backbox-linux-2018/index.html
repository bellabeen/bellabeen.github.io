[tutorial] Cara Membuat Hotspot di Backbox Linux 2018

#Tutorial #Backbox


Yo selamat malam semua nya, pada kesempatan kali ini saya akan memberikan tutorial bagaimana cara nya membuat Access Point atau Hotspot pada Laptop pribadi, atau pada OS Backbox Linux, Debian, Ubuntu. Software yang kita gunakan pada kesempatan kali ini adalah create_ap

Daripada bingung bagaimana mendingan kita lihat tutorial dibawah ini

Install packet dibawah ini
sudo apt-get install dnsmasq haveged hostapd build-essential

Download packet create_ap 
git clone https://github.com/oblique/create_ap 
cd create_ap 
sudo make install
Kita jalankan Service create_ap

service create_ap start

To check the status run:
service create_ap status

Using the systemd features (it is not available on Ubuntu 14.04):
sudo systemctl start create_ap.service
sudo systemctl enable create_ap.service

Memulai membuat Access Point

To create your access point run:

  1. Cek device mana yang akan dibuat sebagai ap dengan perintah ifconfig
  2. Lalu kita jalankan create_ap
sudo create_ap wlan0 wlan0 MyAccessPoint MyPassPhrase

 wlan0 wlan0 adalah wifi adapter yang terdapat pada pc kalian
 MyAccessPoint adalah nama access point yang kalian buat
 MyPassPhrase adalah password yang akan kalian butuhkan untuk konek ke AP *gak wajib pastikan 8 karakter ya
