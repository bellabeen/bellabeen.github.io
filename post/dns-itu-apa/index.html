DNS itu apa?

#Info #Networking #Internet
DNS-BNTechno

Sesuai janji saya sebelum nya ,setelah saya bahas tentang bind tidak lengkap rasa nya bila nggak ngebahas tentang DNS
Apa sih itu dns? apa kegunaan nya? dan jenis jenisnya pasti itu yang ada dipikiran kalian sekarang hehe
DNS adalah singkatan dari Domain Name Server, yaitu adalah sebuah sistem yang menyimpan semua informasi data dari domain ataupun hostname dalam sebuah jaringan. Nah tanpa adanya DNS, maka domain tidak bisa mentranslate atau menerjemahkan domain ke IP Address.

Dulu, pada jaringan menggunakan file HOSTS.TXT untuk mapping/memetakan alamat domain. Sistem ini mempunyai kekurangan, yaitu jika IP address berubah, maka file tersebut juga harus diubah. Nah maka dari itu munculah DNS, jika ada yang mengganti host disatu tempat, maka yang lain akan mengikuti dan mempelajari secara dinamis.

Jenis jenis DNS dan Fungsinya

  • A record , yaitu memetakan hostname ke IP address 32-bit (IPv4).
  • AAAA record , yaitu memetakan hostname ke IP address 128-bit atau IPv6
  • MX Record, ini memetakan domain ke mail exchange server.
  • CNAME Record, kalo yang ini membuat nama alias dari sebuah domain.
  • NS Record, memetakan domain kedalam satu daftar dari DNS Server.

Dan DNS ini digunakan untuk mengatur domain coy, biasanya di control panel domainnya atau juga jika Domain sudah di set Name Server atau NS Recordnya ke sebuah Hostingan, lu bisa setting dari hostingnya, entah itu mau subdomain, mx dan lainnya.

Untuk koneksi internet, kita memerlukan DNS ini. Settingan DNS ada pada konfigurasi Network disetiap komputer. Untuk wifi publik, yang memiliki fungsi DHCP, biasanya DNS sudah diset dari router atau AP.

Jadi Singkatnya, DNS seperti PhoneBook pada handphone, melisting semua nomor hp menjadi nama kontak.
