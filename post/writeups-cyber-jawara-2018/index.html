Bonus 3 (53)

Problem :
Terima kasih telah mengikuti Cyber Jawara. Mohon maaf atas kekurangan yang terjadi.
Soal bisa didownload di sini https://drive.google.com/open?id=1nwnKoMvX9aLHmFHw5pWnPQiheY_HMC9Z
Solved :
File yang berekstensi kan .7z yang merupakan ekstensi yang hampir sama dengan .zip ataupun rar ataupun file archive lain nya
Kita hanya perlu mengekstrak file tersebut, setelah terekstrak didapatkan beberapa file. Disana terdapat file berekstensi .exe

Hmm kami curiga, lalu kami mulai menganalisa satu persatu file .exe tersebut didapatkan bahwa file tersebut berekstensi PNG dan JPG/JPEG. Kami rubah ekstensi file tersebut kemudia didapat lah flag nya pada foto photo_2018-06-15_16-52-11.png

Flag : CJ2018{bonussss_untuk_yang_setia_sama_CJ}



Kategori FORENSIC

Sniffing (50)

Problem :
Anda memiliki rekaman paket data jaringan. Sepertinya ada yang mencoba login menggunakan HTTP request yang tidak terenkripsi. File bisa didownload sini https://ctf.jawara.idsirtii.or.id/files/2b460be8b7d689214a91bfbcdb324b5b/sniffing.pcapng

Solved :
Kita gunakan perintah strings sniffing.pcapng | grep CJ2018
Maka didapatkan flag nya CJ2018%7Bsniffing_is_child_play%7
Tapi perlu di perbaiki sedikit menjadi {CJ2018%7Bsniffing_is_child_play}

Kategori REVERSE ENGINEERING

Hidden Config (50)

Problem :
Terkadang, konfigurasi program diletakkan langsung pada program tersebut dan sebenarnya dapat dilihat dengan mudah.
Anda sebagai hacker tentunya bisa menemukan konfigurasi tersembunyi di program Linux ini bukan?
Download file disini https://ctf.jawara.idsirtii.or.id/files/23011d98b8f050648291cc7088889cb1/hidden_config

Solved :

Kita gunakan perintah strings hidden_config | grep “CJ2018”
