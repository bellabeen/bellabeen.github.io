[android] System UI rasa Zenfone ASUS Galaxy Young

July 22, 2018


System UI Galaxy Young kamu gitu gitu aja? Standar? Biasa? Gak keren?
Disini gua akan kasih nih bagaimana cara nya mempercantik tampilan System UI lu berasa seperti ZenUI milik nya asus
Bahan bahan nya gan:

  1. SGYZenfoneUI-ByArd
  2. VodcaUI
  3. Jangan lupa backup dulu gan
  4. Kopi dan cemilan gan biar gak tegang
Gimana gan cara nya?
  1. Extrak file VodcaUI Terus flash lewat CWM gan
  2. Reboot
  3. Santai dulu gan jangan tegang
  4. Terus flash SGYZenfoneUI lewat CWM kembali
  5. Rasakan sensasi nyaa gan 
Simple kan? Tapi pasten
