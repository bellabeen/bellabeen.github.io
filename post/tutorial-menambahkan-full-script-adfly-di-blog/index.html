Tutorial menambahkan full script adfly di blog

April 13, 2015



Dari pada bengong nungguin selesai libur dikarenakan kelas 3 UN mending kita coba liat dulu tutorial dari saya ini :D
Adfly siapa yang gak tau website ini website yang menyediakan sebagai pemendek url untuk mendownload sebuah file yang disimpan pada web penyimpanan file seperti 4shared,mediafire,dropbox dsb yang akan memendekkan url yang sangat panjang dan akan memendekan nya contoh nya http://bn-techno.blogspot.com menjadi seperti ini http://adf.ly/1BjmSl dan akan mendapatkan uang dari pemendekan url tersebut wihh enak kan :D

Dengan banyak nya url yang ingin kita pendekan ,mendingan kita memasukan script full adfly nya agar kita tidak usah repot repot memendekan nya lagii :D
Gimana itu gan cara nya?
Mending Cek duluu 

1. Masuk dulu ke akun adfly yang telah anda daftar kan
2. Masuk kedalam menu tools / tab tools
 3. Terbuka tab baru dan pilih 'full page script'

 
 4.Setelah anda klik 'full page script' akan terbuka halaman yang berisi kode script,di situ ada dua kode terserah anda mau menggunakan yang mana,sesuaikan saja dengan blog/website anda.disini saya beri contoh dengan memakai script nomer dua (seperti gambar dibawah)

 
 
5. Untuk yang saya beri garis merah silahkan ganti dengan URL blog anda.(http://........com) example: http://bn-techno.blogspot.com


  
Cara pemasangan script nya seperti ini

1. Login kedalam blog anda 
2. Copy kode script yang anda pilih tadi login ke blog dan pilih template.klik edit HTML cari kode  </body> letakkan kode script adfly di atasnya.lihat contoh dibawah ini

 </script><script type='text/javascript'> var adfly_id = 4456714; var adfly_advert = &#39;int&#39;; var exclude_domains = [&#39;http://bn-techno.blogspot.com&#39;]; </script> <script src='https://cdn.adf.ly/js/link-converter.js'/> </body>
3. Ganti yang berwarna merah dengan url blog anda.dan coba di tes dulu dengan melihat blog anda dan mengklik link yang ada seperti readmore atau link ke artikel anda yang terdapat di blog anda

Selamat anda sekarang sudah berhasil memasang full script adfly pada blog anda :D

Hargai penulis jika ingin copas sertakan link blog ini! Kritik dan komentar anda sangat membantu untuk membangun blog ini menjadi lebih baik lagi :) arigatou
