[java] Program True Or False Perbandingan Nilai di Java

September 11, 2018


Selamat semua nya disini gua akan memberikan tau kan sebuah program sederhana pada java bagaimana sih program tersebut membandingkan suatu nilai true or false dari suatu operasi penjumlahan sederhana
Oke kita langsung saja cek bagaimana sih cara buat nya

  1. Pastikan JDK dan JRE sudah terinstall di pc kalian , kalau belum bisa baca postingan ini
  2. Buka editor IDE sesuai selera bisa netbean, sublime,notepad, nano, vim, pico dll 
  3. Lalu ketikan program dibawah ini
import java.util.Scanner;
public class comparasi_nilai{
//Methode Main
public static void main(String[] args) {
Scanner input = new Scanner (System.in);
int bil1, bil2;
int operasi, pertanyaan;
     
//Masukkan bilangan yang ingin dijumlahkan
System.out.print("Program Komparasi nilai true or false"+'\n');
System.out.print("-------------------------------------"+'\n');
System.out.print("Masukkan bilangan pertama : ");
bil1 = input.nextInt();
System.out.print("Masukkan bilangan kedua   : ");
bil2 = input.nextInt();
//Pertanyaan hasil
System.out.print("Berapa hasil penjumlahan dua bilangan diatas ? : ");
pertanyaan = input.nextInt();
operasi = bil1+bil2;
//benar atau salah
if (pertanyaan == operasi) {
System.out.println("Selamat, Anda Benar!");
}
else
{
System.out.println("Maaf,jawaban anda salah!");
System.out.println("Hasil sebenarnya adalah : "+operasi);
}
}
}

Keterangan
 Pastikan comparasi_nilai sama dengan nama file yang akan kalian simpan nanti misal comparasi_nilai.java
     

           4. Simpan file kalian dengan ekstensi .java
           5. Kalian bisa mengkompile di terminal dengan perintah javac comparasi_nilai.java
           6. Lalu jalankan dengan perintah java comparasi_nilai
