[tutorial] Menghubungkan Beberapa PC dengan switch di GNS 3

April 2, 2018

GNS3-BN

Pada postingan saya kali ini saya akan memberitahukan cara menghubungkan beberapa pc dengan switch. Cek dimari


1.      Buka GNS3 nya dan buat file baru dengan nama 2 switch
2.      Buat sebuah topolgi seperti gambar dibawah ini
3.        Hub semua pc dan switch dengan kabel,kalau sudah tekan tanda hijau diatas untuk memulai menjalankan topologi tersebut
4.      Untuk menjalankan dan masuk keterminal pada pc virtual nya dengan klik kanan pada pc lalu pilih console

5.      Lalu konfigurasi ip nya disini saya menggunakan ip berdasarkan npm saya yaitu 18 untuk network nya. Konfigurasi nya dengan perintah ip 192.168.18.15 255.255.255.0Kalau sudah ketikan kembali perintah save untuk mengsave konfigurasi nya
Lalu lanjutkan pada pc yang lain nya hingga selesai


6.      Jika sudah semua di konfigurasi pastikan dahulu apakah konfigurasi telah dibuat dengan perintah show ip

7.      Kita coba pastikan apakah pc kita sudah terhubung satu sama lain dengan perintah ping . Disitu saya melakukan ping dari pc 4 ke pc 1 dan berhasil. Saya mencoba melakukan ping kembali dari pc4 ke pc 3 dan berhasil
