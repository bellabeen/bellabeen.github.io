[java] Cara Koneksi Java Netbean Dengan Database MySQL

#Java #Tutorial
bn-techno

Pada kesempatan kali ini gua akan memberikan codingan untuk mengkoneksikan program java netbean dengan database. Disini saya menggunakan MySQL untuk database nya
Emang gak bisa make Oracle Maria DB dll? Bisa, tapi sekarang saya hanya menggunakan database MySQL
Gimana sih cara nya?

Buat file baru dengan nama koneksi.java
Lalu tambahkan library mysql dengan klik kanan pada file koneksi.java lalu add library mysql

//Connect DB
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class connect {
    private String url,usr,pwd,db;
    connect() {
        db  ="tugas";
        url ="jdbc:mysql://localhost/" +db;
        usr = "root";
        pwd = "";
    }
    Connection getConnect() {
        Connection cn = null;
        try {
            //load Driver Database
            Class.forName("com.mysql.jdbc.Driver");
            cn = DriverManager.getConnection(url,usr,pwd);
            System.out.println("Koneksi berhasil");
        }
        catch (ClassNotFoundException er) {
        }
            catch (SQLException er) {
                    System.out.println("Error #2: " +er.getMessage());
                    System.exit(0);
                    }
        return cn;
            }
    public static void main (String[] args)
    {
        new connect().getConnect();}
        }

Disini kita menggunakan library .sql untuk mengkoneksikan antar program kedatabase nya
Lalu menggunakan driver database yang bernama com.mysql.jdbc.Driver
Untuk bagian ini
connect() {
        db  ="tugas";
        url ="jdbc:mysql://localhost/" +db;
        usr = "root";
        pwd = "";
    }
Disesuaikan dengan database yang kalian buat, disini saya menggunakan database bernama tugas dengan username root dan password tidakada
Karena kita menggunakan library SQLException berguna untuk menangani error handling apabila terjadi kesalahan dalam pengetikan saat pembuatan program
