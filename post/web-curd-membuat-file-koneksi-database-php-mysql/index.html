[web] CURD Membuat File Koneksi Database PHP MYSQL

June 15, 2019




Hallo selamat malam semua nya pada kesempatan kali ini gua akan memberi tahukan cara menggubungkan php dengan database mysql
dari pada berbasa udah pasti basi mending kita cek

  • Buka editor favorit kalian, disini saya menggunakan sublime text
  • Buat folder dengan nama include
  • Lalu didalam folder include buat file dengan nama koneksi.php
  • Lalu masukkan kodingan nya berikut

<?php
$host = "localhost";
$user = "root";
$pass = "kepo";
$db = "asdos";

$connect = mysqli_connect($host, $user, $pass) or die("Koneksi ke database Gagal!");
mysqli_select_db($connect, $db) or die("Tidak ada database yang dipilih");
?>
*keterangan
host = server local, bisa diganti dengan ip dari vps atau hosting kalian
user = username yang digunakan untuk login ke phpmyadmin
pass = password yang digunakan untuk login ke phpmyadmin jika tidak ada password gunakan ''
db = nama database yang akan digunakan
  •  Kalau sudah save dengan ektensi .php 
  • Taru di folder htdocs kalau di windows , kalau di linux taruh di /var/www/html/namafolder
  • Contoh nama folder nya belajar-curd
