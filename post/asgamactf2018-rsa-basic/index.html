Public_key.pem

Setelah kita download, kita mulai mencari private key nya didalam public key.pem dengan menggunakan openssl
openssl rsa -noout -text -inform PEM -in public_key.pem -pubin
Didapat modulus dan exponent

 Karena hasil modulus masih berupa hexadesimal, kita jadikan desimal dahulu  dengan pyton
 

Setelah ketemu hasil nya maka kita factorisasi kan hasil desimal tadi di factordb.com

 Terlihat disana kita menemukan p dan q nya
Kalau sudah kita mulai masukan ke script python

import gmpy2

def num_to_str(num):
    res = “”
    while num > 0:
        res = chr(num % 256) + res
        num = num / 256
    return res

c = 13328445333056206565801100615758037997272587858304987765122445734130
n = 15719648961151124406259408275130518526692619002644355904409352031187
p = 3852454912858673504993326758109153
q = 4080423863932134851980426919817779
e = 65537
t = (p-1)*(q-1)
d = gmpy2.invert(e,t)
m = pow(c,d,n)

print “Flag: “, num_to_str(m)
