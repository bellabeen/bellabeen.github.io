On occasion this time I will provide basic using nmap tools
Nmap an open source tool for exploration and network security audit. It is designed to check large networks quickly, although it can also work against a single host. Nmap uses raw IP packets in a sophisticated way to determine which hosts are available on the network, what service (application name and version) is provided, what operating system (and version) is used, what type of firewall / packet filter is used, and a number of other characteristics

Maybe this is some basic command using nmap

1. Checking the open ports
nmap <host / target IP>
2. Check the specific port on the target machinenmap -p <host / target IP>
3. Checking the service running on the portnmap - sV <host / target IP>
4. Check the target machine port in 1 network segmentnmap <host / target IP>
5. Identify machine operating system
    nmap -O <host / target IP>
