How to split pdf file on Backbox Linux

Pada kesempatan kali ini , gua akan memberikan tutorial bagaimana sih cara nya memisahkan halaman pada file pdf di Backbox Linux, Debian, Ubuntu, dan sejenis nya

Berawal dari keresahan saya untuk mengambil beberapa bagian dalam file pdf, disitu saya menemukan tools yang menurut saya sangat berguna. Daripada banyak cincong mending kita liat aja tutorial nya dibawah ini

Install software pdftk
sudo apt-get install pdftk

Split halaman pdf

pdftk contohsumber.pdf cat 1-5 output contohtarget.pdf <enter>
