CTF?
CTF?
WTF

Adalah salah satu kompetisi dibidang Security. Pada kompetisi ini peserta ditugaskan untuk mencari dan menemukan sebuah string atau disebut "FLAG" . Untuk menemukan flag tersebut peserta bebas menggunakan tools atau metode masing masing .
Dilarang KERAS men DDOS SERVER SOAL!!!

CTF dibagi menjadi 3 kompetisi
  1. Jeopardy
  2. Attack - Defense
  3. Hack Quest
Ada beberapa kategori soal pada soal ctf antara lain  :

  1. Steganography
  2. Programming
  3. Forensics/Analysis
  4. Reversing
  5. Exploit
  6. Misc
  7. Mobile
  8. Pwn 
  9. Trivia 
Tiap jenis soal mempunyai skor dan level pengerjaan yang berbeda pula ,jadi jangan heran kalau menemukan soal yang agak membingungkan hati dan pikiran

APA SIH MANFAAT NYA??
Menguji sampai sejauh mana Skill dan Ability kalian atau kalian yang merasa diri jago dalam dunia IT atau kalian yang merasa diri sebagai seorang hacker, cracker, atau apalah - apalah. Disamping itu, kalian juga bertambah wawasan dan pengalaman dalam memecahkan masalah (Problem Solving)

Kira kira apa sih yang harus dipelajari?

Web
Pengenalan mengenai web perlu seperti sql,html,pytho,javascript,protocol dll yang berguna mencari jawaban pada soal web

Crypthography
Mulai lah mempelajari dari caesar, rot13, xor, vigenere, scaytale, base64, base 32,md5 dll kita juga bisa menggunakan tools ini cryptii.com/ salah satunya

Steganography
 Bagaimana menganalisa sebuah data atau gambar, biasanya soal steganography diberikan dalam bentuk gambar,file musik,video bahkan


Forensic
Pelajari lah bagaimana mengekstrak data dari sebuah file. File bisa berupa disk image, bisa berupa network capture. Dalam hal ini, kita perlu tahu mengenai berbagai jenis filesystem. Untuk forensic yang berhubungan dengan networking, kalian perlu memahami banyak protokol dasar, seperti misalnya HTTP, FTP, HTTPS, dsb. Cara memahami protokol - protokol nya bisa menggunakan tools ini https://www.wireshark.org/download.html


Reversing dan exploit
Mempelajari bahasa C dan Assembly, kalian juga perlu menggunakan tool IDA Pro (windows) untuk mengetahui alur program

Misc
Gabungan dari beberapa jenis soal yang saling berkaitan

Mobile
Pelajarilah bagaimana melihat data pada file backup pada android

PWN
Soal nya hampri sama dengan Reversing bedanya dalam jenis soal reversing kode diberikan kepada kita dan dijalankan di mesin kita. Kalau Pwn, kode dijalankan di dalam mesin remote dan flag yang harus kita ambil, berada didalam mesin remote tersebut biasa nya kita mengakses menggunakan putty

Trivia
Diperlukan kemampuan menggunakan Search Engine istilah kerennya Google-fu.

Kiat lain nya kalian bisa melihat event event ctf di website ctftime.org
