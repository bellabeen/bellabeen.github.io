Yoo whatssapp guys, pada kesempatan kali ini gua akan memberikan tutorial bagaimana sih cara nya menginstall android version oreo di Zenfone 3 Max , Karena gua juga pemakai ASUS gua bagi nih tips untuk Install Versi Oreo di Zenfone 3 Max  ZC553KL & ZC553TL

Versi android 8.1 ini yang diberi nama Orea ini sangat masih. Versi yang dirilis setelah merilis versu 7.0 Nougat. Versi ini mempunyai fitur baru yaitu notifikasi baru untuk aplikasi, mode gambar dalam gambar, instalasi aplikasi lebih compatible, sangat baik untuk copy paste, sangat cepat untuk scene yang berjalan di background, sangat baik untuk daya hidup batrai dll

Widih keren kan? Lalu
Bagaimana cara mengecek Update OTA secara manual?
Dengan masuk ke  menu setting -> about phone -> Software update
Pastikan batrai terisi minimal 50% da pastikan koneksi internet anda sangat stabil, karena proses download nya sangat lama

Download Firmware
Untuk mengantisipasi kalian bisa download disini itu untuk versi 8.1 Oreo
Pastikan backup data kalian untuk mengantisipasi

Cara install
Ada beberapa methode yang digunakan disini kita gunakan metode recovery
Install via Recovery
  • Download semua bahan nya, kemudia rename menjadi update.zip
  • Masuk ke recovery mode dengan menekan power button + volume up tombol
  • Pilih apply update from sd card
  • Pilih file update.zip  yang telah kamu download sebelum nya
  • Tunggu hingga proses selesai
  • Nanti Zenfone kamu reboot sendiri  
  • Rasakan sensasinya
