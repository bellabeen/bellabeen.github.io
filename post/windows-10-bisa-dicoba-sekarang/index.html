Windows 10 bisa dicoba sekarang

December 23, 2014

Beberapa hari  setelah diperkenalkan, Microsoft akhirnya memberikan akses bagi para pengembang dan antusias Windows untuk mencicipi sistem operasi baru Windows 10 disitus resminya

Mulai hari kamis (2/10/2014), mereka dapat mengunduh versi awal Windows 10Technical Preview melalui halaman preview.windows.com. Proses pendaftaran ke Windows Insider Program dibutuhkan terlebih dahulu untuk bisa mengunduhWindows 10 Technical Preview.

Pendaftaran hanya membutuhkan data nama, alamat e-mail, password, dan negara calon pengguna. Dengan mengikuti program beta tester tersebut, mereka akan selalu mendapatkan update dan fitur terlebih dahulu.

“Tidak ada jaminan bahwa apa yang Anda dapatkan adalah menjadi versi finalnya, namun anda bisa berkontribusi menentukan seperti apa wujud Windows 10nantinya dengan memberikan kritik dan masukan,” demikian kata Microsoft.
