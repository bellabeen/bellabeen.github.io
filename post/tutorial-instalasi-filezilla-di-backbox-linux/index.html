[tutorial] Instalasi FileZilla di Backbox Linux

November 2, 2018

 

Karena banyak yang bertanya bagaimana sih instalasi ftp client di backbox, pada kesempatan kali ini saya akan memberikan tutorial bagaimana cara instalasi ftp client pada Backbox Linux, Debian Latest Version, Ubuntu 16.10 Latest Version
Disini saya menggunakan FileZilla untuk mentransfer file dari client ke server web
Daripada bingung mending langsung aja kita cuss

  • Buka terminal yang sering kalian gunakan 
  • Ketikkan perintah sudo apt-get update 
  • Ketikkan lagi perintah sudo apt-get install filezilla 
  • Kalau sudah tunggu hingga proses selesai
 
  • Tinggal Buka FileZilla nya yang sudah terinstall dan FileZilla siap digunakan
 
Kurang lebih tampilan FileZillanya seperti itu , tidak jauh beda dengan di windows
