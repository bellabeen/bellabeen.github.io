[tutorial] Cara Mengubah Hostname di Linux Debian, Xubuntu, DLL

#Linux #Tutorial #Cara

Hostname adalah nama komputer, yang mana ketika kita menginstall linux kita diminta untuk memasukkan nama komputer. Hostname muncul di terminal setelah tanda @ (@hostname)
Ada beberapa cara mengganti nama hostname, salah satu nya menggunakan perintah linux atau melalui terminal
Cara ini bisa diterapkan di distro debian, ubuntu, xubuntu, backbox, kali linux dan distro turunan lainnya

APA SIH HOSTNAME di LINUX?

Nama yang diberikan untuk sebuah komputer pada jaringan. Hostname sebagai penanda komputer pada sebuah jaringan


Foto diatas menunjukkan username nya hachi dan hostname nya geek
Hostname di identifikasi secara unik pada jaringan komputer. Hostname terdiri dari angka, huruf, karakter tanda hubung dan tanda periode. Hostname terdiri dari 2 - 63 karakter

Kita bisa menggunakan perintah pada terminal
sudo nano /etc/hostname
 


Ubah sesuai keinginan anda. Untuk selesai melihat perubahan anda harus merestart komputer untuk melihat hasil nya
