[tutorial] Instalasi Android Studio pada Parrot OS Linux

#Linux #Tutorial #parrot #Android
Selamat pagi gaess pada kesempatan kali ini gua akan memberikan sebuah tutorial yang sangat bermanfaat
Tutorial pada kesempatan kali ini mengenai Tutorial Instalasi Android Studio pada Parrot OS. Tidak ada spesial tentang cara menginstall nya, mungkin dari kalian masih bingung maka dari itu gua akan memberitahukan tutorial singkat nya


Download terlebih dahulu android studio nya di web resmi nya https://developer.android.com/studio/
Kita bisa lihat spesifikasi minimum instalasi. Karena disini saya menggunakan sistem operasi linux dalam instalasi nya , maka pilih untuk linux 64bit . Saya menggunakan Parrot Linux dengan Based Debian dengan system requirement
Linux
  • GNOME or KDE desktop
Tested on Ubuntu® 14.04 LTS, Trusty Tahr (64-bit distribution capable of running 32-bit applications)
  • 64-bit distribution capable of running 32-bit applications
  • GNU C Library (glibc) 2.19 or later
  • 3 GB RAM minimum, 8 GB RAM recommended; plus 1 GB for the Android Emulator
  • 2 GB of available disk space minimum,
    4 GB Recommended (500 MB for IDE + 1.5 GB for Android SDK and emulator system image)
  • 1280 x 800 minimum screen resolution
Kalau untuk windows atau MAC OS kalian  bisa lihat sendiri, system requirement yang digunakan. Download seperti biasa, kalua sudah extrack file tersebut. Lalu masuk kedirektori android studio/bin/  


maka ditemukan file studio.sh lalu eksekusi file tersebut dengan ./studio.sh
 


 






Kita memilih apakah kita akan mengimport settingan sebelum nya iya atau tidak, disini saya memilih do not import settings karena tidak ada settingan sebelum nya
