[info] Beberapa plugin wajib Editor Atom

October 23, 2018


Karena banyak diluar sana yang memperdebatkan mana yang lebih bagus
Apakah sublime?
Apakah Atom?
Karena saya menggunakan kedua nya untuk editor programming jadi tidak ada yang dipersalahkan

Hanya saja dari kedua editor tersebut di custome sesuai kebutuhan menggunakan plugin yang sudah banyak dikembangkan oleh developer
Mungkin saya akan menshare beberapa plugin yang kudu wajib bin harus untuk dipakai di editor Atom

Emmet
Plugin yang digunakan untuk mempermudah penulisan tag html dan css secara tepat dan cepat
Instalasi bisa dilihat di http://emmet.io/download/

JS Snippet
Plugin yang digunakan untuk meminimalkan pemakaian kode secara berulang dengan code javascript , dan node JS
Instalasi bisa dilihat di https://atom.io/packages/javascript-snippets

CSS Comb
Plugin yang digunakan untuk mempercantik tampilan source code dari css kita, semakin rapih semakin enak juga dilihat
Instalasi bisa dilihat di https://atom.io/packages/atom-csscomb

JS Hint
Plugin yang digunakan untuk mengkoreksi dan memvalidasi source code JS sesuai aturan. Dan mempermudah mencari bug dan error di javascript
Instalasi bisa dilihat di https://atom.io/packages/atom-jshint

File Icons
Plugin yang digunakan untuk mengurutkan dan menambah ikon pada filename berdasarkan file type.
Instalasi bisa dilihat di https://atom.io/packages/file-icons

Git Plus
Plugin yang digunakan agar kita bisa bekerja dengan git, baik itu untuk commit, checkout, push/pull, diff dan perintah git lainnya tanpa menutup text editor atom.
Instalasi bisa dilihat di https://atom.io/packages/git-plus

Beauty

Plugin yang digunakan untuk  membuat kode anda kelihatannya berantakan tampil lebih cantik dan mudah dibaca.
Instalasi bisa dilihat di https://atom.io/packages/atom-beautify
