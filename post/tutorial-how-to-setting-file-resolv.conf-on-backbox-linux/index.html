[tutorial] How to Setting File Resolv.conf on Backbox Linux

September 6, 2018

Sedikit cerita aja, beberapa waktu yang lalu gua iseng menginstall DNS Server (bind9) , Dhcp Server, dan Proxy Server (squid) pada Backbox Linux
Setelah itu laptop tidak gua hidupkan selama seharian , pas pagi nya gua coba konekin ke Internet kampus dannn bumm secara otomatis semua akses internet gua gak bisa dijalankan
Saat itu panik sekali, setelah itu cari cari lahh problem nya di forum forum tapi tidak ada artikel yang relevan bagaimana cara setting file resolv.conf di backbox linux
Setelah telusuri ehh permasalahan nya ada pada file resolv.conf nya

Disana hanya ada

nameserver 127.0.0.1
Wahh curiga gini gua, setelah gua coba edit edit file itu ehh ternyata bisa guyss
Edit  nya seperti ini
sudo nano /etc/resolv.conf
Tambahkan nameserver nya
nameserver 8.8.8.8
nameserver www.google.com
Setelah itu kalian save

Jangan lupa agar file itu tidak berubah saat direstart maka kita edit juga pada file /etc/resolvconf/resolv.conf.d/base untuk membuat permanent

Dengan perintah
sudo nano /etc/resolvconf/resolv.conf.d/base
Jangan lupa tambahkan nameserver nya
nameserver 8.8.8.8
 nameserver www.google.com
