Program Konversi Sederhana Waktu C++/C Plus Plus


Ini merupakan codingan program sederhana bagaimana kita mengkonversi waktu , mulai dari detik, menit, dan jam atau sebalik nya
Disini saya menggunakan geany sebagai editor, compiler gcc dan os Backbox Linux
Dari pada banyak cakap mending langsung cek codingan nya

#include <iostream>
using namespace std;

float jam, menit, detik, ss, mm, hh;
char konversi;
long selection;
int main(){

cout<<"Program Sederhana Konversi Waktu\n\n";
cout<<"1. Konversi Jam"<<endl;
cout<<"2. Konversi Menit"<<endl;
cout<<"3. Konversi Detik"<<endl<<endl;
cout<<"Masukan Menu : ";cin>>selection;
   switch(selection){
case 1 :
        cout<<"Konversi Jam ke Menit dan Detik "<<endl<<endl;
        cout<<"Masukkan jumlah Jam : ";cin>>hh;
        cout<<endl;
   
    jam = hh;
    menit = hh*60;

    cout<<"Jam ke jam : "<<jam<<endl;

    jam = mm/60;
    menit = mm;
    detik = mm*60;
   
    cout<<"Menit ke jam : "<<jam<<endl;
    cout<<"Menit ke menit : "<<menit<<endl;

case 3 :
    cout<<"Konversi Detik ke Menit dan Jam"<<endl<<endl;
