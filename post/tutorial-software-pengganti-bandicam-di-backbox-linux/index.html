Open Broadcaster Software


Software keren nih kan , wajib hukum nya buat kalian semua cobain. Karena gua user Backbox Linux disini gua akan memberikan tutorial bagaimana cara nya menginstall itu software di Backbox Linux. Eittsss tenang Distro lain nya nanti gua kasih tau yaa :p
Disini gua memakai Backbox Linux yang based on Ubuntu 16.04 xenial , untuk versi > 16.04 bisa juga sudah teruji
Pertama
Buka terminal andalan kalian dan ketikan
sudo add-apt-repository ppa:obsproject/obs-studio
sudo apt-get update
sudo apt-get install obs-studio

Kedua
Tunggu hingga proses selesai, kalau sudah selesai. Mari kita setting

Ketiga
Software akan terbuka seperti dibawah ini
 
Lalu pada bagian kiri bawah pada tab sources kita klik tanda tambah lalu pilih  Video Capture untuk menampilkan hasil record dari webcam kita 



Lalu klik tambah lagi lalu pilih Screen Capture untuk merecode layar laptop kita
Kalau sudah kalian bisa sesuai kan dengan selera , dengan menambahkan record dari speaker ataupun headset,earphone, maupun lain nya

Cukup mudah kan?
Arch Linux
Dengan menggunakan perintah
sudo pacman -S obs-studio
Atau langsung menggunakan git di AUR
“Git” version is available on AUR: https://aur.archlinux.org/packages/obs-studio-git
Manjaro
Dengan menggunakan perintah
sudo pacman -S obs-studio

Untuk distro lain nya kalian bisa mengunjungi github nya
https://github.com/obsproject/obs-studio/wiki/Install-Instructions#linux-install-directions
