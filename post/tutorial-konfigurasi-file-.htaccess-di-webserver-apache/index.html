[tutorial] Konfigurasi file .htaccess Di Webserver Apache

November 1, 2018

Yoo selamat semua nya, pada kesempatan kali ini gua akan memberitahukan cara nya bagaimana sih mengkonfigurasi file .htaccess pada direktori sebuah website
Ehhh bang emang .htaccess itu file apa?
Kalian bisa baca postingan gua sebelum nya disini
Daripada banyak cincong mending langsung aja kita konfigurasi

  1. Pastikan buat file bernama .htaccess pada directory
  2. Lalu buka file tersebut menggunakan text editor yang disukai
  3. Disini saya menggunakan script Options -Indexes, apa sih gunanya script Option -Indexes ? Script tersebut berfungsi agar orang lain tidak bisa mengakses index directory file pada website kita
  4. Bisa di lihat di website demo saya
  5. Kalian bisa mengakses http://bellabeen.000webhostapp.com/aboutme/ kalian bisa lihat perbedaan nya 
  6. Masih script lain nya seperti
order allow,deny //untuk memblock dari ip address tertentu
# This will deny the IP
172.15.23.9 deny from 172.15.23.9
# This will deny all IP's from 172.15.23.0 through
172.15.23.255 deny from 172.15.23

ErrorDocument 404 /404.html //untuk membuat file 404 not found

Redirect 301 /main.html /test1/index.html //untuk redirect ke halaman tertentu

order deny,allow //untuk mengizinkan mengakses dari ip address tertentu
# Denies all IP's Deny from all
# This will allow the IP
172.15.23.9 allow from 172.15.23.9
# This will allow all IP's from 172.15.23.0 through 172.15.23.255
allow from 172.15.23

Untuk melihat konfigurasi file lain nya bisa dilihat di file documentasi dari apache
