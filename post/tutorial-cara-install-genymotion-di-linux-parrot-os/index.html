[tutorial] Cara Install Genymotion di Linux Parrot OS

#Linux #Tutorial #parrot


Hii guys pada kesempatan kali ini gua akan memberikan tutorial instalasi emulator android, emulator kali ini gua menggunakan emulator Genymotion. Kenapa menggunakan Genymotion? Karena mungkin genymotion yang support di Linux. Ada banyak emulator yang tersebar cuman tidak support dengan linux

Disini gua menggunakan Linux Parrot OS dalam instalasi nya

Pertama
Spesifikasi pc atau laptop kalian harus sesuai system requirment
  1. Ubuntu 18.04 (Bionic beaver) - 64bit
  2. Debian 9 (Stretch), Fedora 29 - 64bit
  3. 64 bit CPU, with VT-x or AMD-V capability, enabled in BIOS settings
  4. Hardware accelerated GPU
  5. 400 MB disk space
  6. 2GB RAM
Kalian buat akun terlebih dahulu di https://www.genymotion.com/account/login/ 

Kalau sudah kalian bisa login terlebih dahulu
Download genymotion nya sesuai arsitektur pc atau laptop kalian bisa 64bit atau 32bit

Kedua
 Ekstrak file genymotion kalian, dengan perintah
unzip namafile.zip

Ketiga
Masuk ke folder genymotion nya dengan perinth cd genymotion/

Keempat
Eksekusi file genymotion nya dengan perintah ./genymotion
Lalu proses instalasi akan jalan dengan sendiri
Kelima
Pilih device yang akan kalian pakai apa itu google nexus dll sesuaikan dengan kebutuhan. Disini gua menggunakan device Google Nexus 7
