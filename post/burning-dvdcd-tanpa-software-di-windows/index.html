Burning DVD/CD Tanpa Software di Windows

November 1, 2015


Pada awal nya penyimpanan data kedalam DVD atau pun CD populer dikalangan para pengguna komputer sebagai pengganti Disket yang penyimpanan data nya hanya sedikit kapasitas nya

Teknologi pun sekarang telah berkembang sangat pesat dan media penyimpanan data telah berkembang dengan sangat banyak seperti Flasdisk,maupun HDD Eksternal yang sedang berkembang sekarang,tapi terkadang kita harus menggunakan CD maupun DVD untuk mengirim data kepada orang lain

Sudah tau kan apa itu burning? Belum nihh Heheh :D
Oke gua kasih tau deh
Menurut gua Burning itu adalah teknik membakar data kedalam CD maupun DVD yang menggunakan Software tertentu untuk membakar file tersebut

Tapi kali ini saya akan memberi tau ngeburn tanpa menggunakan software apapun itu
Gimana sih cara nya?

Oke kita simak tutorial dibawah ini

  1. Masukkan DVD maupun CD kedalam DVD ROM 
  2. Lalu secara otomatis akan muncul gambar seperti ini,isikan Disk Title tersebut terserah kamu
  3. Pilih With CD/DVD Player klik Next
  4. Pilih dan pindahkan file kedalam CD/DVD yang akan di burn
  5. Kalau sudah terdapat file didalam CD/DVD tersebut,klik kanan pada drive DVD tersebut lalu pilih BURN  TO DISC,terus next aja
  6. Selanjutnya proses burning akan berjalan secara otomatis
  7. Tunggu hingga proses tersebut selesai
