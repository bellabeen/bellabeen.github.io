[tutorial] Cara cepat install Paket .deb di Parrot OS

#Linux #Tutorial #parrot

Mungkin dari kalian semua mengalami kesulitan apabila menginstall paket aplikasi di linux
Pada dasar nya instalasi paket aplikasi yang berekstensi .deb menggunakan perintah
dpkg -i namaaplikasi.deb

Mungkin itu terlalu rumit ya walaupun hanya sebaris perintah, kalau di ubuntu kita bisa menggunakan ubuntu software center tapi menggunakan itu sangat menghabiskan resource yang sangat banyak, disini saya menyarankan menggunakan aplikasi Gdebi 
Whats apa itu gdebi?
Gdebi merupakan tools yang bisa membantu kalian untuk menginstall aplikasi berekstensi .deb


Instalasi Gdebi on Parrot

Gdebi bisa diintsall di turunan linux based on Debian. Karena disini saya menggunakan Parrot OS yang based on Debian jadi bisa dipraktikan

Oke kita coba install Gdebi nya
 
Dengan perintah
sudo apt-get install gdebi

Menjadikan GDebi Sebagai Default Installer

Setelah GDebi terinstal sekarang saatnya menjadikan GDebi sebagai tools default untuk menginstal file .deb.
  • Anda harus memiliki file .deb, download saja Google Chrome, itu sudah ada versi .deb-nya.
  • Nah selanjutnya, klik kanan pada file .deb, lalu klik properties.
  • Di properties Anda akan melihat pilihan Open With, ganti itu dengan GDebi.
Selanjutnya jika Anda ingin menginstal file .deb Anda hanya perlu double klik pada file .deb yang ingin diinstall.
