[tutorial] Install Netbeans on Backbox Linux

#Java #Tutorial #Backbox

bellabeen

Hallo guyss , setelah kemarin kalian sudah melihat bagaimana cara menginstall Java JDK di Linux nahhhh sekarang gua bakalan ngasih tau nih bagaimana cara nya menginstall IDE Netbeans di Backbox Linux
Kok kenapa harus Backbox sih kak? Kenapa gak Ubuntu, Debian atau linux lainnya?
Yaaaa karena gua user BackBox Linux :p

Daripada terlalu lama mendingan cussss

Cek Version Java 

Dengan perintah
javac -version

Download Netbeans

cd /tmp && wget -c http://download.netbeans.org/netbeans/8.2/final/bundles/netbeans-8.2-linux.sh 
Tunggu hingga download nya selesai

Install Netbeans

Setelah didownload packet nya, kita tidak perlu berpindah ke direktori lain tetap di direktori /tmp saja
chmod +x netbeans-8.2-linux.sh

sudo ./netbeans-8.2-linux.sh
Setelah itu maka akan  terunning sendiri
Kita sesuaikan konfigurasi nya sesuai selera
