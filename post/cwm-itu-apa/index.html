CWM itu apa?

#Info #Internet #Android

Huahh sudah malem nih semua nya,tapi apa mau dikata harus update ini blog :D
Kali ini saya akan memberitahu sedikit tentang CWM yang seneng ngoprek harus punya ini karena hukum nya wajib bangett lohh
Gak mau kan data dalam hp ilang apalagi sms darii si itu tuhh wkkwwk :p

ClockWorkModatau CWM adalah sebuah recovery pada perangkat Android, yang secara bentuk fungsi dan penerapannya hampir serupa dengan recovery default pada android kebanyakan. Biasanya pada recovery default menu yang tersedia adalah:
  1. Reboot system now = Untuk reboot Android
  2. Apply update from sdcard = untuk memilih zip dalam SD Card
  3. Wipe data/factory reset = untuk reset data data Android 
  4. Wipe cache partition = untuk menghapus cache dalam android
Itulah tampilan menu dalam recovery default, tidak seperti recovery default pada  CWM terdapat 8 menu pilihan yang sangat membantu sobat dalam berbagai hal, antara lain
  1. Apply Update.zip = sobat bisa root HH sobat dari sini
  2. BackUp and restore = disinilah kalian bisa  memback-up, segala data-datanya termasuk ROM default sebelum dioprek akan tersimpan dan utuh layaknya sebelum melakukan instalasi ROM ketika di restore.
  3. Mount and storage= penjelasan nya bisa digoogle ya :p
  4. Advanced = dipilihan ini terdapat banyak lagi pilihan yang tersedia, salah satunya partisi SD Card.
Aplikasi ini bisa dibilang adalah modding atau modifikasi dari recovery mode yang terdapat pada perangkat dengan sistem operasi Android. Biasanya, para pengguna yang suka mengutak atik /oprek perangkatnya menggunakan mode recovery ini untuk menginstall atau meng-custom beberapa hal yang berkaitan dengan sistem, misalnya menginstall dan mengganti jenis font pada perangkatnya, menambahkan fitur quick panel (panel pengaturan praktis) pada status bar, menginstall script -script tertentu, mengubah icon - icon ,ganti custom rom dan masih banyak lagi.
 
Peran CWM menjadi sangat penting dalam urusan installasi custom ROM (cusrom). Karena untuk dapat menginstall ROM yang telah dimodifikasi oleh para sesepuh, pengguna memerlukan ClockWorkMod Recovery sebagai “alat” untuk menginstall. Itu artinya, tanpa adanya mode recovery ini pengguna tidak akan bisa memasang custom ROM pada perangkat Android yang dimilikinya.
Cara menginstall CWM sendiri bisa berbeda - beda tiap merek / tipe perangkat yang digunakan, namun yang pasti syarat utama yang dibutuhkan untuk dapat memasangnya adalah perangkat harus sudah di root. Selain untuk menginstallasi beberapa hal yang saya sebutkan diatas, CWM juga bisa berperan sebagai mode recovery layaknya mode recovery bawaan perangkat, yakni bisa digunakan untuk membackup data pada perangkat kita, serta me-restore (mengembalikan) nya jika suatu ketika kita membutuhkannya, serta mengembalikan perangkat ke pengaturan awal (factory reset atau kembalikan ke pengaturan pabrik).
