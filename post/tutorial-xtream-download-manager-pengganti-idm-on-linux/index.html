Hallo gaes , kalau yang baru baru migrasi ke linux pasti masih kurang nyaman nih menggunakan linux, apalagi para downloader yang setiap hari kudu wajib download apapun , anime , film, software dan yang berbau ++

Jangan risau cuyy, dilinux juga ada loh software kaya IDM di weedus
Apa sih itu?

Yaps Xtream Download Manager merupakan salah satu software open source untuk downloader di linux, sebenernya banyak software lain
Mungkin lain kesempatan kita bahas

Xtream Download Manager merupakan software powerfull yang speed up download up to 500% , bisa menyimpan streaming video dari youtube, facebook,google video dll
Fitur yang dikasih juga gak kalah menarik, ada fitur resume downloads, broken/dead link downoad , penjadwalan download dan masih banyak lain nya

Kira kira gimana sih menginstall nya?
Ada dua cara instalasi nya

Pertama langsung lewat website nya
Link download disini

Kedua langsung tambahkan repository kita


sudo add-apt-repository ppa:noobslab/apps
sudo apt-get update
sudo apt-get install xdman

Tampilan nya seperti gambar diatas
