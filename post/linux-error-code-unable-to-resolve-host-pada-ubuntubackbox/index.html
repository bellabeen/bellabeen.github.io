[linux] Error code unable to resolve host pada Ubuntu,Backbox

#Debian #Linux #Tutorial #Ubuntu #Info #Networking #solved #Backbox

Pada beberapa hari yang lalu saat  setelah menginstall linux backbox pada pc pribadi terjadi error yang aneh

code error tersebut
Error message 'sudo: unable to resolve host <USER>'
Setelah analisa pada hari sebelum nya saya mengganti nama hostname pada linux saya , dikarena kan mengganti hostname tersebut terjadi error pada linux saya saat melakukan akses pada root. Disana terjadi perbedaan antara hostname dan hosts pada linux saya

Kita pastikan nama pada hostname dan host terjadi kesamaan

hachi@geek:/etc/network$ cat /etc/hostname
geek
hachi@geek:/etc$ cat hosts
127.0.0.1    localhost
127.0.1.1    geek
Kita bisa mengedit hostname dengan perintah
hachi@geek:/etc$ sudo nano /etc/hostname
Kita bisa mengedit host dengan perintah
hachi@geek:/etc$ sudo nano /etc/hosts
127.0.0.1       localhost
127.0.1.1       geek

# The following lines are desirable for IPv6 capable hosts
::1     ip6-localhost ip6-loopback
fe00::0 ip6-localnet
ff00::0 ip6-mcastprefix
ff02::1 ip6-allnodes
ff02::2 ip6-allrouters

Kode tersebut bisa dipakai pada ubuntu, debian, backbox, kali linux dll 
Demikian tutorial singkat saya semoga bermanfaat
