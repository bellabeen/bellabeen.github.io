Apa itu Proxy??

#Pengetahuan #Internet



Hallo selamat pagi sobat blogger..
Kali ini saya akan menjelaskan apa sih itu proxy dan apa keuntungannya untuk kita


Yo so pasti,udah jangan banyak ngemeng yang penting cekidot...


 Proxy terbagi menjadi  2 yaitu teknik proxy dan proxy server

Teknik proxy
Teknik yang digunakan untuk mengakses internet secara bersama sama oleh beberapa komputer sekaligus dalam sebuah jaringan LAN melalui,sebuah modem,atau sebuah saluran komunikasi

Proxy Server
sebuah sistem yang berperan sebagai perantara antara client hosts, dengan server yang ingin diaksesnya. - See more at: http://belajar-yok.blogspot.com/2008/10/apa-yang-dimaksud-dengan-proxy-server.html#sthash.2KpQ2wOw.dpuf
 Sebuah sistem yang berperan sebagai perantara antara client host dengan server yang ingin diakses nya.Proxy server juga bertindak sebagai gateway terhadap dunia maya untuk setiap komputer klien.Web server yang menerima request dari proxy server akan menginterpretasikan request-request tersebut seolah-olah request itu datang secara langsung dari komputer klien, bukan dari proxy server. Proxy server juga dapat digunakaan private network yang dihubungkan ke sebuah jaringan publik

So,untungnya proxy apa?
 Mempercepat prose layanan
Nah maksud nya itu gimana? Gini loh,saat komputer merequest ke halaman web google.com melalui proxy,halaman itu akan disimpan oleh proxy ke halaman cache.Tujuan nya agar suatu saat ente mau merequest halaman yang sama.proxy tidak perlu lagi mengambil nya secara langsung ke komputer server nya google langsung.Proxy cukup mengambil dari cache tersebut
