May 9, 2018


FIREWALL PADA GNS3

Banyak cara nya ada yang membuat firewall pada gns3 berdasarkan ip address,berdasarkan ip network, berdasarkan port , berdasarkan protokol dll
Disini kami mendrop atau membuat firewall (memblok semua koneksi) berdasarkan ip address sumber

Setting ip pada router dengan perintah
ip address add address=192.168.43.97⁄24 interface=INTERNET
ip address add address=192.168.2.1⁄24 interface=ether2


Setelah ip tersetting
Kita masukkan firewall nya dengan perintah
ip firewall filter add chain=input src-address=192.168.43.97 action=accept
ip firewall filter add chain=input src-address=192.168.2.2 action=drop


Setting ip pada pc
Dengan perintah ip 192.168.2.2⁄24 255.255.255.0 192.168.2.1
Lalu kita coba ping ke router apa kah berhasil? Kalau timeout berarti settingan anda berhasil membuat firewall
