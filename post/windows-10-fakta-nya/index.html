Try It Harder

Muhammad Bella Buay Nunyai | 09-04-2015

Windows 10 fakta nya

#Info #Internet


Alih alih banyak yang menantikan  Windows 9, eh ternyata secara mengejutkan  Microsoft malah merilis Windows 10 alih-alih Windows 9 sebagai penerus Windows 8.1. Sistem operasi tersebut ditujukan untuk digunakan di berbagai perangkat, dari mobile, desktop, hingga untuk menjalankan server.

Banyak yang bertanya-tanya mengapa Microsoft seolah “melompati” dari Windows 8 langsung ke Windows 10. Menurut Cnet (30/9/2014), dengan penamaan yang melompati satu versi itu, Microsoft mencoba menunjukkan keseriusannya dengan memberikan perubahan yang secara signifikan  di sistem operasi baru dikembangkan.

Banyak juga pertanyaan-pertanyaan lain di dalam benak pengguna, seperti kapan mereka bisa menggunakannya, bagaimana dengan aplikasi-aplikasinya, serta pertanyaan seputar antarmuka dan tombol Start Menu,dan fitur yang ditawarkan oleh pihak Microsoft

Berikut adalah rangkuman singkat tentang fakta-fakta Windows 10 yang dirangkum dari situs Cnet.
Upgrade pada sistem operasi nya

Pengguna OS Windows bisa mendapatkan sistem operasi Windows 10 diwebsite resmi microsoft secara beta yang masih banyak kemungkinan bug yang ditemukan . Microsoft berencana melucurkan sistem operasi ini setelah ajang konferensi tahunan Build bagi para pengembangnya. Kemungkinan Windows 10 meluncur di pertengahan atau akhir tahun 2015 semoga sesuai jadwal yang telah ditentukan pihak pengembang

Namun, jika mereka ingin merasakan lebih dahulu sistem operasi Windows 10 ini, mereka bisa mendaftarkan dirinya dalam Windows Insider Program yang mulai dibuka pada Rabu (1/10/2014) waktu AS. Pendaftaran bisa dilakukan di situs Microsoft

Dengan mengikuti program ini, maka pengguna bisa mendapatkan salinan sistem operasi tahap awal (kemungkinan masih banyak bug/versi beta). Microsoft juga belum memutuskan berapa harga versi retail untuk OS Windows 10 (pasti mahal ini yang original nya :D)

Aplikasi yang ditawarkan

Belum banyak hal yang dibagikan oleh Microsoft tentang aplikasi dalam Windows 10 yang menghebohkan ini. Namun sejauh ini, aplikasi-aplikasi yang akan diluncurkan di Windows 10 bakal bersifat universal, alias bisa dijalankan untuk semua jenis perangkat, mulai dari PC, smartphone, hingga tablet bisa dibilang alih alih hampir mirip dengan Windows 8

Karena itu, developer diwajibkan untuk mengcustomisasi aplikasi buatannya agar bisa berjalan di semua platform sistem operasi windows, mulai dari segi tampilan hingga bagaimana caranya berjalan di semua perangkat dengan beragam ukuran layar dan resolution yang ditawarkan. Aplikasi-aplikasi tersebut bisa diunduh melalui Windows Store yang telah di ditawarkan fitur dari pihak microsoft

Antarmuka pengguna 

Berbeda dengan sistem operasi Windows 8 dimana aplikasi yang dimiliki sepenuhnya dioperasikan dengan layar sentuh, maka di Windows 10 ini sifatnya opsional hampir sama dengan fitur yang ditawarkan windows 7

Saat pengguna memasangkan perangkat tabletnya dengan sebuah docking keyboard, maka antarmuka yang ditampilkan berupa tampilan desktop standar komplit dengan tombol Start yang kembali hadir.

Begitu docking keyboard dilepas, maka pengguna akan mendapatkan antarmuka layar sentuh layaknya sistem operasi Windows 8.

Dengan fitur antarmuka Snap, pengguna juga bisa menampilkan lebih dari satu aplikasi dalam satu jendela. Microsoft menambahkan dukungan kuadran layout dimana pengguna bisa menampilkan empat jendela aplikasi sekaligus yang berjalan bersamaan.

Terdapat juga dukungan untuk beragam antarmuka desktop. Dengan demikian, pengguna bisa beralih dari satu antarmuka aplikasi yang sedang berjalan ke antarmuka desktop dengan mudah.

Tombol Start menu
 
Seperti disinggung di atas, tombol Start menu telah dihadirkan kembali oleh Microsoft di layar desktop-nya layaknya seperti windows 7. Tombol tersebut kembali menempati posisi sakralnya selama ini, di pojok kiri bawah :D *kaya udah sepuh aja

Namun yang berbeda kali ini, tampilan Start Menu yang diusung Microsoft dalamWindows 10 merupakan gabungan dari Windows 7 dan Windows 8.

Selain menampilkan daftar shortcut ke berbagai aplikasi, Start menu tersebut juga memiliki bagian yang menampilkan jendela Live Tiles, yaitu menu kotak-kotak khas Windows 8.

Dukungan perangkat yang ditawarkan

Belum pasti apakah semua perangkat Windows saat ini yang beredar dipasaran akan mendukung Windows 10 atau tidak. Microsoft mengembangkan Windows 10 dari basis Windows 8.1 yang telah mengusung komputasi 64-bit maka spek buat 32-bit memungkin kan tidak akan bekerja tapi bisa saja nanti diantisipasi dari pihak microsoft nya dikemudian hari

Dengan demikian, beberapa prosesor lawas tidak lagi didukung oleh sistem operasi baru Microsoft ini. Namun untuk PC, Windows 10 akan bisa dijalankan dalam sistem yang belum terlalu lama.
Kesimpulannya, jika kita membeli perangkat Windows yang diluncurkan pada tahun 2014 hingga satu tahun mendatang, besar kemungkinannya perangkat tersebut bisa dipasangi Windows 10 :D
Berikut ini adalah detail spesifikasi minimum Windows 10 Preview dari pihak Microsoft
  • Processor: 1 gigahertz (GHz) or faster
  • RAM: 1 gigabyte (GB)
  • Free hard disk space: 16 GB
  • Graphics card: Microsoft DirectX 9 graphics device with WDDM driver
  • A Microsoft account and Internet access
Sebelumnya pernah muncul kabar bahwa Windows 10 Preview ini hanya bisa diinstall di processor Intel saja. Namun hal tersebut sepertinya sudah tidak relevan lagi bagi pengguna AMD. Microsoft nampaknya telah merilis versi preview yang bisa berjalan di processor Intel maupun AMD.
So..pengguna AMD tidak perlu ragu lagi, kamu bisa menginstall Windows 10 Preview sekarang :D
