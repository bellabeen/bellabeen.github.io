Program Sederhana C Plus Plus / C++

Pada kesempatan kali ini saya akan memberikan contoh tutorial  kodingan program sederhana mencari luas balok dengan penggunaan Pointer di Program C Plus Plus / C++

Kita bisa lihat pada kodingan berikut

#include <iostream>
using namespace std;

    void luas (int p, int l, int t) {
        int l_ps ,ptr;
        l_ps = 2((p*l)+(p*t)+(l*t));
        ptr = &l_ps;
cout<<“Luas Bangun     = “<<l_ps <<endl;
        cout<<”=================================\n”;
        cout<<“Letak Pointer\n”;
        cout<<“*ptr = “<<*ptr<<endl;
        cout<<“ptr = “<<ptr<<endl;
        cout<<“&l_ps= “<<&l_ps<<endl;
    }

int main () {
    int pj, lb,tb;

cout<<“Program Menghitung Luas Balok\n”;
cout<<“Masukan panjang = “;cin>>pj;
cout<<“Masukan lebar   = “;cin>>lb;
cout<<“Masukan tinggi  = “;cin>>tb;
luas(pj, lb, tb);
return 0;
}
Kita bisa eksplorasi program tersebut untuk menghitung luas atau volume bangun ruang maupun bangun datar dengan menggunakan program C Plus Plus / C++
